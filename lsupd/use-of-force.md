# 👊 ขั้นตอนการใช้กำลัง (Use of Force)

เพื่อรักษาบรรยากาศ Roleplay และความเหมาะสมในบริบทมหาวิทยาลัย เจ้าหน้าที่ต้องปฏิบัติตามลำดับขั้น (Continuum) ดังนี้:

| ระดับ | การตอบโต้ของเจ้าหน้าที่ | วิธีปฏิบัติ / อุปกรณ์ | สถานการณ์ตัวอย่าง |
| :--- | :--- | :--- | :--- |
| **Level 1** | **การปรากฏตัว (Presence)** | จอดรถเปิดไฟไซเรน / ยืนคุมเชิง | วัยรุ่นรวมกลุ่มซิ่งรถ, พ่นกำแพง |
| **Level 2** | **คำสั่งวาจา (Verbal)** | สั่งให้หยุด/วางของ (สุภาพ) | สั่งให้หยุดพฤติกรรม, สั่งให้แยกย้าย |
| **Level 3** | **ควบคุมกายภาพ (Soft)** | ใส่กุญแจมือ, ล็อคแขน, กดจุด | พยายามเดินหนี ขัดขืนแต่ไม่สู้คน |
| **Level 4** | **ตอบโต้ด้วยกำลัง (Hard)** | ชก, เตะ, ทุ่ม, สเปรย์พริกไทย | เริ่มโจมตีเจ้าหน้าที่ด้วยมือเปล่า |
| **Level 5** | **อาวุธไม่ถึงชีวิต (Less-Lethal)** | กระบอง, ปืนไฟฟ้า (Taser) | มีอาวุธประชิด (มีด/ไม้), ขับรถหนี |
| **Level 6** | **กำลังขั้นเด็ดขาด (Lethal)** | อาวุธปืนประจำกาย (Lethal Weapon) | ใช้ปืนยิงต่อสู้, พยายามฆ่า |

### ⚠️ กฎเหล็ก
* **ห้ามใช้กำลัง** หากผู้เล่นยอมปฏิบัติตามคำสั่งวาจา (Level 2)
* การใช้กำลังต้อง **ทัดเทียม** กับภัยคุกคามเสมอ
