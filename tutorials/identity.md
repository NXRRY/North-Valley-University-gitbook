# 🤝 ระบบการทำความรู้จัก (Identity Interaction)

ในเซิร์ฟเวอร์ของเรา คุณจะเริ่มต้นด้วยการเป็น **"คนแปลกหน้า" (Stranger)** ของทุกคน เพื่อส่งเสริมการ Roleplay ที่สมจริง คุณจำเป็นต้องทำความรู้จักกับผู้อื่นก่อนจึงจะเห็นชื่อของพวกเขา

### 🔍 วิธีการทำความรู้จัก
คุณสามารถเลือกใช้งานได้ 2 วิธี (ขึ้นอยู่กับระบบที่ถนัด):

#### วิธีที่ 1: ผ่านระบบสายตา (Target System)
1. กดปุ่ม **`ALT (Left)`** ค้างไว้แล้วเล็งไปที่บุคคลที่ต้องการ
2. เลือกเมนู **"แนะนำตัว / ทำความรู้จัก" (Introduce Yourself)**
3. เมื่ออีกฝ่ายกดยืนยัน ชื่อของเขาจะปรากฏแทนที่คำว่า "Stranger"

#### วิธีที่ 2: ผ่านคำสั่งแชท (Command)
* **`/meet [ID]`** : ส่งคำขอทำความรู้จักไปยังผู้เล่นตามหมายเลข ID ที่ระบุ
* **`/checkid`** : ตรวจสอบ ID ของคนรอบข้างในระยะใกล้ (เพื่อใช้คู่กับคำสั่ง /meet)

---

### 📇 ผลลัพธ์หลังการทำความรู้จัก
* **Nametags:** ชื่อของผู้เล่นคนนั้นจะปรากฏบนหัวเมื่อคุณเข้าใกล้
* **Contact:** ระบบจะบันทึกเบอร์โทรศัพท์ของเขาลงในเครื่องของคุณอัตโนมัติ (หากระบบโทรศัพท์รองรับ)
* **Memory:** ระบบจะจดจำข้อมูลนี้ไว้แม้คุณจะออกเกมไปแล้ว (Persistent Identity)

---
> [!IMPORTANT]
> **มารยาทในการทำความรู้จัก:**
> การทำความรู้จักควรเกิดขึ้นผ่านบทบาท (In-Character) เสมอ การเดินไปรัวคำสั่งใส่ผู้อื่นโดยไม่พูดคุยก่อนอาจถือเป็นการรบกวนบรรยากาศการเล่น (Immersion Break)
