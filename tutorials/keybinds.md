# ⌨️ ปุ่มกดและคำสั่งพื้นฐาน

เพื่อให้การใช้ชีวิตในแคมปัสลื่นไหลที่สุด นี่คือคู่มือปุ่มกดลัดที่สำคัญสำหรับการควบคุมตัวละครและระบบต่างๆ ภายในเซิร์ฟเวอร์

***

### 🎮 การควบคุมพื้นฐาน (Essential Controls)

| ปุ่มกด    | ฟังก์ชันการใช้งาน                 | ระบบที่เกี่ยวข้อง   |
| --------- | --------------------------------- | ------------------- |
| **`F1`**  | เปิดเมนูวงกลม (Radial Menu)       | Supreme Radial      |
| **`TAB`** | เปิดกระเป๋าสัมภาระ (Inventory)    | QB-Inventory        |
| **`M`**   | เปิดโทรศัพท์มือถือ (Phone)        | QB-Phone            |
| **`ESC`** | เมนูพักเกม / ตั้งค่า (Pause Menu) | Codem-Pause         |
| **`P`**   | เปิดแผนที่ (Map)                  | Codem-Pause         |
| **`Z`**   | เปิดเมนู Radial เสริม / ดู Hotbar | ox\_lib / Inventory |

***

### 🚗 ยานพาหนะและการเดินทาง (Vehicle & Travel)

| ปุ่มกด          | ฟังก์ชันการใช้งาน                   | ระบบที่เกี่ยวข้อง |
| --------------- | ----------------------------------- | ----------------- |
| **`L`**         | ล็อค/ปลดล็อค ยานพาหนะ               | Vehicle Keys      |
| **`B`**         | คาดเข็มขัดนิรภัย (Seatbelt)         | Jim-Mechanic      |
| **`F`**         | ขึ้น/ลง รถ (Hold เพื่อเลือกที่นั่ง) | Jim-Mechanic      |
| **`L-Shift`**   | ใช้งาน NOS (Boost)                  | Jim-Mechanic      |
| **`L-Control`** | เปลี่ยนโหมด NOS                     | Jim-Mechanic      |
| **`Y`**         | ทิ้งสมอเรือ (Anchor Boat)           | NX-Parking        |

***

### 🎭 ท่าทางและการสื่อสาร (Emotes & Voice)

| ปุ่มกด           | ฟังก์ชันการใช้งาน               | ระบบที่เกี่ยวข้อง |
| ---------------- | ------------------------------- | ----------------- |
| **`H`**          | ยกมือขึ้น (Hands Up)            | Emote Menu        |
| **`B`**          | ชี้มือ (Point Finger)           | Emote Menu        |
| **`U`**          | แกล้งตาย / ล้มลง (Ragdoll)      | Emote Menu        |
| **`X`**          | ยกเลิกท่าทาง / ยกเลิกงานที่ทำ   | Emote / ox\_lib   |
| **`F5`**         | เปิดเมนูท่าทาง (Emote Menu)     | Emote Menu        |
| **`ALT (Left)`** | ใช้สายตาจ้องมอง (Target System) | QB-Target         |
| **`ALT (Left)`** | คุยวิทยุ (Radio Talk)           | PMA-Voice         |
| **`~ (Grave)`**  | เปลี่ยนระยะการพูด (Proximity)   | PMA-Voice         |

***

### 🎒 ระบบกระเป๋าและไอเทม (Inventory Slots)

| ปุ่มกด      | ฟังก์ชันการใช้งาน                     |
| ----------- | ------------------------------------- |
| **`1 - 5`** | ใช้งานไอเทมตามช่องที่ตั้งไว้ (Hotbar) |
| **`Z`**     | กดค้างเพื่อดูสถานะไอเทมใน Hotbar      |

***

### 🎓 คำสั่งสำหรับเจ้าหน้าที่และนักศึกษา (Special Menus)

* **`F7`:** เปิดเมนูระบบใบแจ้งหนี้ (Billing Menu)
* **`F10`:** เปิด/ปิด ชื่อบนหัว (Nametags)
* **`F11 - F12`:** เมนูสำหรับทีมงาน (Staff/Dev Menu)

***

> #### 💡 คำแนะนำเพิ่มเติม
>
> * หากปุ่มกดมีปัญหา หรือต้องการเปลี่ยนปุ่ม สามารถไปที่: **ESC > Settings > Key Bindings > FiveM**
> * หากต้องการยกเลิกการกระทำที่กำลังทำอยู่ (Progress Bar) ให้กดปุ่ม **`X`**
