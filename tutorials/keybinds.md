# ⌨️ ปุ่มกดและคำสั่งพื้นฐาน (Controls & Commands)

เพื่อให้การใช้ชีวิตในแคมปัสลื่นไหลที่สุด นี่คือคู่มือปุ่มลัดและคำสั่งที่สำคัญสำหรับการควบคุมตัวละครและระบบต่างๆ ภายในเซิร์ฟเวอร์

---

### 🎮 การควบคุมพื้นฐาน (Essential Controls)

| ปุ่มกด | ฟังก์ชันการใช้งาน | ระบบที่เกี่ยวข้อง |
| :--- | :--- | :--- |
| **`F1`** | เปิดเมนูวงกลม (Radial Menu) | Supreme Radial |
| **`TAB`** | เปิดกระเป๋าสัมภาระ (Inventory) | QB-Inventory |
| **`M`** | เปิดโทรศัพท์มือถือ (Phone) | QB-Phone |
| **`ESC`** | เมนูพักเกม / ตั้งค่า (Pause Menu) | Codem-Pause |
| **`P`** | เปิดแผนที่ (Map) | Codem-Pause |
| **`Z`** | ดู Hotbar / สถานะไอเทม (กดค้าง) | ox_lib / Inventory |
| **`ALT (Left)`** | ใช้สายตาจ้องมอง (Target System) | QB-Target |

### 💬 คำสั่งทางช่องแชท (Chat Commands)

เซิร์ฟเวอร์ของเราใช้ระบบคำสั่งเพื่อช่วยในการดำเนินเนื้อหา Roleplay ให้สมจริงและมีมิติยิ่งขึ้น

#### 🎭 คำสั่งสำหรับการ Roleplay (RP Commands)
| คำสั่ง | ระยะ (เมตร) | คำอธิบายการใช้งาน |
| :--- | :--- | :--- |
| **`/me`** | 10.0 | แสดงการกระทำ (เช่น * สมชาย ยื่นมือไปรับปากกา) |
| **`/do`** | 10.0 | บรรยายสถานะรอบตัว (เช่น (( มีกลิ่นควันจากรถ )) ) |
| **`/ame`** | 10.0 | แสดงการกระทำบนหัว (ไม่ขึ้นในช่องแชท) |
| **`/say`** | 10.0 | การพูดคุยปกติในระยะใกล้ |
| **`/w`** | 2.0 | **กระซิบ:** ได้ยินเฉพาะคนที่อยู่ชิดกันเท่านั้น |
| **`/l`** | 5.0 | **พูดเบา:** ลดระยะการได้ยินลง |
| **`/s`** | 40.0 | **ตะโกน:** ส่งเสียงให้ได้ยินในระยะไกล |

#### 📻 การสื่อสารและระบบส่วนตัว (Social & Info)
* **`/radio [ข้อความ]`** : สื่อสารผ่านวิทยุ (ข้อความปรากฏตามเลขช่องวิทยุ)
* **`/b [ข้อความ]`** : **Local OOC** คุยเรื่องนอกบทบาทในบริเวณใกล้เคียง (50 เมตร)
* **`/ooc [ข้อความ]`** : **Global OOC** ประกาศข้อความนอกบทบาททั้งเซิร์ฟเวอร์
* **`/charinfo`** : ตรวจสอบข้อมูลเบื้องต้นของตัวละคร
* **`/cash` / `/bank`** : ตรวจสอบจำนวนเงินสด และ เงินในธนาคาร

#### 🚔 คำสั่งเฉพาะอาชีพ (Job Commands)
* **`/lspd [ข้อความ]`** : ประกาศข่าวสารทางการจากหน่วยงานตำรวจ (แสดงผล UI พิเศษ)

---

### 🎓 เมนูพิเศษ (Special Menus)

* **`F7`:** เปิดเมนูระบบใบแจ้งหนี้ (Billing Menu)
* **`F10`:** เปิด/ปิด ชื่อบนหัว (Nametags)
* **`F11 - F12`:** เมนูสำหรับทีมงาน (Staff/Dev Menu)

---

> [!TIP]
> **คำแนะนำเพิ่มเติม:**
> * หากต้องการเปลี่ยนปุ่มลัด: **ESC > Settings > Key Bindings > FiveM**
> * หากต้องการยกเลิกการกระทำ (Progress Bar): กดปุ่ม **`X`**
> * การใช้คำสั่ง `/ooc` หรือ `/b` ควรใช้เมื่อจำเป็นเท่านั้น เพื่อรักษาบรรยากาศการเล่น (Immersion)
